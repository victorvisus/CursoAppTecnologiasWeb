HTML básico y conceptual

Se trata no sólo de dar estructura aparente al documento, sino de organizarlo internamente con una lógica que permita su mantenimiento (poscionamiento, SEO, correcciones evolutivas, o escalabilidad) a largo plazo.

Cuestiones de accesibilidad y usabilidad

Para el usuario es esencial una experiencia satisfactoria de uso y acceso al contenido. En desarrollo se ha pensado e implementado la forma adecuada para este fin: estructuración del documento, jerarquía entre contenidos, uso de elementos que ordenan lo expuesto, etc.

Introducción al uso de estilos

Por ejemplo a un texto como LOREM IPSUM: Lorem ipsum dolor sit amet consectetur adipiscing elit. Quisque faucibus ex sapien vitae pellentesque sem placerat. In id cursus mi pretium tellus duis convallis. Tempus leo eu aenean sed diam urna tempor. puede tener un estilo en cursiva, y con letra gótica, un color gris, ...

Ordenación de elementos de modo jeráquico

El orden de importancia de los elementos debe corresponderse con: indentaciones correctas en codificación, visión de la empresa hacia sus clientes o usuarios, facilidad para la comprensión del lector promedio, etc.
